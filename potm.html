Problem of the Month

Monthly prompt with fun awards



The Problem of the Month (PotM) is a monthly server competition where participants answer an open-ended prompt. The best submission receives Discord Nitro, and all winners get special server roles.


Join the OPS server
Make sure you're in the server (join here), and check out #monthly-question.
Submit your response in the form
Read the monthly question, open the form, and submit.
Awards
Top submission gets 1 month of Discord Nitro, courtesy of the OPS admins. The winner and runner-ups will get special server roles.
